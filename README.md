# C-Panel
Building a Control Panel comparable to a streamdeck completely from scratch.
Using Fusion for the casing design and pcb design. The Casing is going to be printed on a BambuLab A1 Mini and the pcb is going to be produced by PCBWay.
Technical basis is  Rapi Zero H to control all functions of the Panel.
Everything else is going to be figured out in the Way.
